agroforestgrid

Settings Export / Import About agroforestgrid

agroforestgrid is used for planning the layout of agroforestry plantations.

  • Start by zooming and panning the map to the desired location.
  • Create a new project (plus icon) and give it a name.
  • Draw an area on the map using the polygon tool.
    • There are two types of polygon; Boundary & Tree Group
    • Boundary is used to outline the boundary of a segment.
    • Tree Groups are used to outline groups of trees within a segment.
    • If no Tree Groups are drawn, trees will be created within the boundary.
  • Draw a grid onto the map (of the configured width & height).
  • Trees can be drawn in two ways:
    • When trees are drawn using the top toolbar, they will appear on the grid intersections points inside the polygons. (Tree Groups or Boundary)
    • Individual trees can be drawn using the circle tool.
  • The number of trees in the segments is displayed at the bottom of the map.
  • Individual trees can be dragged around using the hand tool.
  • Individual trees or polygons can be deleted by double-clicking on them with the hand tool.
  • The Polygon, Grid or Trees can be toggled On/Off or deleted (bin icon).
  • Multiple Segments can be create within each Project.
  • Multiple Tree Groups can be create within each Segment.

A Google Maps API Key is required for this application to function correctly. You can get an API key from here. Enter your API key in the Settings (gear icon) at the top right of this page.

Touch gestures are supported on touch screen devices; one finger pan, two finger pinch zoom, two finger rotate and two finger slide up & down to tilt.
